# Editable Workflow Examples

These examples demonstrate the reversible workflow pattern:

- skills return a `Workflow` with `build(...)`
- workflows can be cloned and overridden
- workflows can be inspected before execution
- workflows can be built from a small YAML graph description

Examples:

- `example_clone_override.py`
- `example_inspect_and_json.py`
- `example_yaml_skill.py`
- `example_wan21_clone_override.py`
